# Visa-tracker
Checks how many days abroad you have left to fulfil your visa conditions

How to use
1. Download and open abroad_days.html using your favourite browser
2. Input any trips you plan to take
3. These can be adjusted in the added trips table
4. if you want to see how many days are used up in a particular 12-month period use the query portion and put in your start date
5. Complaint and not compliant indicate if you meet/dont meet your visa requirements respectively
